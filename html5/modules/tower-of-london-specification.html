Tower of London
Description:
No duration in this task.
Three pins. First pin fits 3 balls, second pin fits 2 ball and third pin fits 1 ball.
Three Balls. Green, Red and Blue.
Same start position of balls in every presentation: 2 moves - Pin 1 - Green, Red, Pin 2 - Blue Pin 3 -
Final position plus number of moves is presented.
Example: 2 moves - Pin 1 - Green, Blue, Pin 2 - , Pin 3 - Red
Test 1: 2 moves - Pin 1 - , Pin 2 - Blue, Green , Pin 3 - Red
Test 2: 2 moves - Pin 1 - Green, Pin 2 - Blue, Pin 3 - Red
Test 3: 3 moves - Pin 1 - Green, Blue, Pin 2 - Red, Pin 3 -
Test 4: 3 moves - Pin 1 - Green, Pin 2 - Blue, Red, Pin 3 -
Test 5: 4 moves - Pin 1 - Red, Green, Pin 2 - Blue, Pin 3 -
Test 6: 4 moves - Pin 1 - Blue, Pin 2 - Red, Green, Pin 3 -
Test 7: 4 moves - Pin 1 - Red, Blue, Pin 2 - , Pin 3 - Green
Test 8: 4 moves - Pin 1 - , Pin 2 - Red, Blue, Pin 3 - Green
Test 9: 5 moves - Pin 1 - Red, Green, Blue, Pin 2 - , Pin 3 -
Test 10: 5 moves - Pin 1 - Red, Blue, Green, Pin 2 - , Pin 3 -
Test 11: 5 moves - Pin 1 - Blue, Green, Pin 2 - Red, Pin 3 -
Test 12: 5 moves - Pin 1 - Blue, Pin 2 - Red, Green, Pin 3 -
When a ball is touched possible moves is shown.
If an attempt is not successful, result is saved and the same presentation is repeated. Three attempts are allowed for each presentation.
